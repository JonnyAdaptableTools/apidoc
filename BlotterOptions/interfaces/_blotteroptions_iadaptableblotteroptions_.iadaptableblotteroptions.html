Interface IAdaptableBlotterOptions

The class injected into the Adaptable Blotter at startup. It forms the ONLY parameter requried by the Adaptable Blotter constructor. It provides all the layout, DataGrid, config and other nformation required to ensure a full user experience. The class contains a few properties of which 2 (vendorGrid and primaryKey) are mandatory - and a number of 'xxxOptions' classes. Any property that is not supplied by the user when populating the object, will use the default value as supplied in 'DefaultAdaptableBlotterOptions'.
